Hệ thống chatbot sử dụng một Mô Hình Điều Phối Trung Tâm để quản lý luồng dữ liệu và điều phối các Agents chuyên biệt. Các agents này là các mô hình nhỏ, mã nguồn mở, được tối ưu hóa cho từng nhiệm vụ cụ thể, giúp xử lý nhanh và hiệu quả mà không cần đào tạo lại.

2. Danh Sách Các Loại Agents

Agent Tiền Xử Lý Đầu Vào

Chức năng: Làm sạch và chuẩn hóa văn bản đầu vào (loại bỏ lỗi chính tả, ký tự thừa, phân tách từ).

Công nghệ: spaCy, regex.

Ví dụ: "Ai phụ trách dự án A???" → "Ai phụ trách dự án A".

Agent Nhận Diện Ý Định

Chức năng: Phân loại ý định của câu hỏi (hỏi thông tin, yêu cầu hành động, v.v.).

Công nghệ: BERT-small, DistilBERT.

Ví dụ: "Ai phụ trách dự án A" → Ý định: "yêu cầu thông tin nhân sự".

Agent Trích Xuất Thực Thể

Chức năng: Xác định các thực thể quan trọng (tên, ngày, địa điểm).

Công nghệ: Flair NER, spaCy NER.

Ví dụ: "Ai phụ trách dự án A" → Thực thể: "dự án A".

Agent Tạo Phản Hồi

Chức năng: Sinh ra câu trả lời tự nhiên dựa trên ý định và thực thể.

Công nghệ: GPT-2 small, DialoGPT.

Ví dụ: Ý định: "yêu cầu thông tin", Thực thể: "dự án A" → "Người phụ trách dự án A là Nguyễn Văn B".

Agent Tích Hợp Dữ Liệu

Chức năng: Truy vấn và tổng hợp thông tin từ các nguồn nội bộ (CRM, ERP).

Công nghệ: API connectors, lightweight DB query models.

Ví dụ: "Dự án A có bao nhiêu người?" → Truy vấn CRM → "Dự án A có 5 người tham gia".

Các agents này hoạt động độc lập nhưng phối hợp chặt chẽ dưới sự quản lý của Mô Hình Điều Phối Trung Tâm, đảm bảo xử lý nhanh và chính xác.

3. Luồng Hoạt Động Của Hệ Thống

1. Đầu Vào: Nhân viên nhập câu hỏi → Gửi đến Mô Hình Điều Phối.
2. Tiền Xử Lý: Agent Tiền Xử Lý làm sạch văn bản.
3. Nhận Diện Ý Định: Agent Ý Định xác định mục đích câu hỏi.
4. Trích Xuất Thực Thể: Agent Thực Thể tìm các thông tin quan trọng.
5. Tích Hợp Dữ Liệu: Agent Dữ Liệu truy vấn thông tin từ hệ thống nội bộ.
6. Tạo Phản Hồi: Agent Phản Hồi sinh câu trả lời tự nhiên.
7. Đầu Ra: Mô Hình Điều Phối gửi phản hồi cuối cùng cho người dùng.

Ví dụ đầy đủ: Nhân viên hỏi: "Ai phụ trách dự án A?"
- Agent Tiền Xử Lý: "Ai phụ trách dự án A".
- Agent Ý Định: "Yêu cầu thông tin nhân sự".
- Agent Thực Thể: "dự án A".
- Agent Dữ Liệu: Truy vấn CRM → "Nguyễn Văn B".
- Agent Phản Hồi: "Người phụ trách dự án A là Nguyễn Văn B".
- Đầu Ra: Hiển thị "Người phụ trách dự án A là Nguyễn Văn B".
